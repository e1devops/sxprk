## Autorole
Autorole can work with more than 1 role. You can add roles to the list, remove from list, toggle autorole thing.
Bla bla bla.
| Command | Description |
|---|---|
| `/joinrole add @role` | Add a role to the autorole list (given to new members on join) |
| `/joinrole remove @role` | Remove a role from the autorole list |
| `/joinrole list` | Show all roles in the autorole list and current status |
| `/joinrole toggle true` | Enable autorole |
| `/joinrole toggle false` | Disable autorole |
